# Predictive Maintenance

Projekt magisterski dotyczacy predykcyjnego utrzymania ruchu.

## Cel projektu

Celem repozytorium jest uporzadkowanie prac zwiazanych z:

- przygotowaniem i analiza danych,
- budowa modeli predykcyjnych,
- ewaluacja jakosci modeli,
- dokumentacja wynikow do pracy magisterskiej.

## Struktura projektu

```text
Predictive Maintenance/
|- data/
|  |- raw/
|  |- interim/
|  |- processed/
|- docs/
|- models/
|- notebooks/
|- reports/
|- src/
|  |- predictive_maintenance/
|     |- data/
|     |- features/
|     |- models/
|     |- visualization/
|- tests/
```

## Szybki start

1. Utworz i aktywuj srodowisko wirtualne.
2. Zainstaluj zaleznosci z `requirements.txt`.
3. Umieszczaj surowe dane w `data/raw/`.
4. Rozwijaj kod badawczy w `src/`, a analizy robocze w `notebooks/`.

## Status

Repozytorium zostalo przygotowane jako punkt startowy do dalszego rozwoju projektu.
