# Centrifugal Pump Assets

This directory is reserved for future SCADA/HMI visuals for the `Centrifugal Pump System` mode.

## Expected formats

- `SVG` for schematic pump layouts
- `PNG` for static panel graphics or fallback images

## Naming convention

- `centrifugal_pump_overview.svg`
- `centrifugal_pump_component_<name>.svg`
- `centrifugal_pump_overlay_<name>.png`

## SVG component identifiers

Future SVG files should use stable element identifiers, for example:

- `motor`
- `shaft`
- `bearings`
- `mechanical_seal`
- `impeller`
- `pump_housing`

This sprint does not add final graphics yet.
