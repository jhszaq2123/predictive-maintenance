# Shared Icon Assets

This directory is reserved for shared SCADA/HMI icons.

## Expected formats

- `SVG` for monochrome or semantic icons
- `PNG` only when vector assets are not practical

## Naming convention

- `icon_status_normal.svg`
- `icon_status_warning.svg`
- `icon_status_critical.svg`
- `icon_navigation_<name>.svg`

Keep icon names descriptive and stable for reuse across pages.
