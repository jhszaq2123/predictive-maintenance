# Industrial Drive Assets

This directory is reserved for future SCADA/HMI visuals for the `Industrial Drive System` mode.

## Expected formats

- `SVG` for schematic machine views
- `PNG` for static overlays or fallback illustrations

## Naming convention

- `industrial_drive_overview.svg`
- `industrial_drive_component_<name>.svg`
- `industrial_drive_overlay_<name>.png`

## SVG component identifiers

Future SVG files should use stable element identifiers for interactive highlighting, for example:

- `electric_motor`
- `coupling`
- `gearbox`
- `bearings`
- `pump`

This sprint does not add final graphics yet.
